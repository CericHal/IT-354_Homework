API Overview

So I bet you are wondering what API's are and why they are important to Web development. API's (application program interface) are a set of routines, protocols, and tools used for building software applications. It allows for applications to communicate with each other, even exchange information. There are many API's out there that perform a variety of functions such as copying and pasting text from one application to another is through an API.

This technology is similar in concept to modules, as it allows you to integrate software components into your code easily to provide higher functionality without doing it from scratch. API's make developers lives so much easier in that regard.


What is REST?

REST is an acronym that stands for REpresentational State Transfer, and is defined as a software architecture style that has six guidelines used to create Web services. To keep it brief, REST must have a client-server interface, the session must be stateless, it must be cacheable, have a uniform interface, a layered system, and lastly code on demand, which is optional.


What is Postman and why use it?

Now you might be wondering what this postman thing is and what does it have to do with APIs. To give you a summary, Postman is an API developer tool, it helps programmers build test and modify APIs. It is a very useful tool as it can make various types of HTTP requests like GET, POST, PUT, and PATCH. Not only that, but it can save environments for later use, and even convert the API to multiple coding languages for you to use like JavaScript for example. Postman saves time, and makes API development easier and simpler for developers which is why it is so popular.
