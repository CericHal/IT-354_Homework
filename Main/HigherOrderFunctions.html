Higher-order Functions

Higher-order Functions are by technical definition is a function that accepts a function as a parameter, a function that returns a function as its technical definition. Basically they are used to help clean up your code so it looks simpler, but conceptually it can be more complex. Advanced Array methods

Picking up from where High-order functions left off, lets talk about advanced array methods. These methods can benefit your code in the same way Higher-order functions can. Here is our array which we be using our methods on.

The forEach array method purpose is too loop through an array without having to type out a for-loop. Take a look at the example below.

Filter

The filter array method is does exactly what its name says. It goes through the array and takes a function parameter that contains the condition you want to filter the array by. Take a look at the example below where we create a new array that contains only super hero movies.

Find and findIndex

The find method is used to locate the first element in the array that satisfies the condition which is being searched for.

The findIndex function is used the same way the find function is, but it returns the index of the first element that satisfies the condition.

Map

Next method is the map array method which is basically like the filter method, it takes a function that goes through the array and returns a modified copy of the the original array.

Flat and flatMap

The flat method is used to create a new copy of the array but with all the sub-arrays elements added to the main array. Flattening it if you will. If you have a triple nested array you need to set the depth (the number between flat method parentheses) to two. Otherwise it will not completely flatten the array.

The flatMap method takes an array, maps it and then proceeds to flatten it.

Reduce

The reduce method is a function that reduces the array into a single value. For example adding all the values in an array to find the total. take the example below.

Sort

Sort method takes in a function that then returns a copy of the array that is sorted by whichever way we want. In our example we are sorting the movie array by it release dates from oldest to newest.

Push, Unshift, Shift, and Pop methods

Push method takes an array and adds a new element to the end of the array and returns the new length.

Unshift method takes the array and adds a new element to the beginning of the array.

Shift method removes the first item of the array.

Pop method is used to remove the last item in the array.
